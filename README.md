# NagiMemo

書くのが楽しくなる、てがろぐのスキンファイル郡です。V4.7.0対応。

現在ベータ版です。てがろぐはアップデートされていくCMSですので、スキン自体の完成は目指していないです。自分が使いやすい用にカスタマイズしたものですので、気ままに更新していきます。

## ライセンスについて
Nagi MemoはMIT LICENSEを適用しています。

Nagi Memo は「自由に使っていいですよ」というライセンスです。
自分のサイトで使うだけなら、特に何もしなくて大丈夫です。
