## 🆙 NagiMemo Updater（一括アップデートツール）
「スキンの更新があるたびに、FTPで何枚も上書きアップロードするのは面倒だな…」という方のために、AIと協力して**ワンクリックで最新版に更新できるツール**を用意しました。

- **ダウンロード**: [nagimemo_update.php](https://cdn.jsdelivr.net/npm/@lichiphen/nagimemo-updater/nagimemo_update.php)
- **導入と使いかた**:
  1. 上記リンクから `nagimemo_update.php` をお手元に保存します。
  2. 保存したファイルをテキストエディタで開き、**パスワード**（必須）や**IP制限**などの設定を自分用に書き換えます。
  3. てがろぐの本体（`tegalog.cgi`）があるフォルダに、このファイルをアップロードします。
  4. ブラウザで `(てがろぐのURL)/nagimemo_update.php` にアクセスし、設定したパスワードを入れるだけで最新版のスキンが自動で適用されます！
